# 🚀 Codex Skill 简易教程（大创版）

---

# 🧠 一、Skill 是什么

Skill = 固定套路 + 自动执行

👉 本质：
- Prompt：临时任务
- Skill：可复用工作流

---

# 🧩 二、创建 Skill 的方式

## 方法1：Codex App

步骤：
1. 打开 Codex App
2. 点击 New Skill
3. 输入说明
4. 保存

---

## 方法2：本地创建（推荐）

### 创建目录

.codex/skills/debug/

### 创建文件

skill.md

### 写内容

# Debug Skill

## Description
用于修复代码问题

## Instructions
1. 分析问题
2. 定位错误
3. 最小修改
4. 解释原因

---

## 方法3：安装现成 Skill

$skill-installer install xxx

---

# 🧠 三、如何调用 Skill

直接写：

使用 debug skill 修复这个问题

或：

让 Codex 自动调用
