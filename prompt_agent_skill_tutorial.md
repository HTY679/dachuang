# 🧠 Prompt / Agent / Skill 完整教程（大创实战版）

---

# 📌 一句话理解

Prompt = 你说的话  
Agent = 帮你干活的AI  
Skill = AI的能力模块  

---

# 🧩 一、Prompt（提示词）

## 1. 什么是 Prompt

Prompt 是你给 AI 的任务说明书。

👉 你描述需求 → AI 执行

---

## 2. Prompt 三个层次

### 基础 Prompt

帮我做一个旅游网站首页

👉 太简单，效果一般

---

### 进阶 Prompt

请帮我生成一个旅游网站首页，使用 React：

- 导航栏
- 轮播图
- 景点卡片
- 页脚

要求：
- 风格简洁
- 代码完整
- 可运行

---

### 高质量 Prompt（推荐）

目标：
新增旅游网站首页

上下文：
React 项目

功能：
- 导航栏
- 轮播图
- 景点展示

限制：
- 不改已有功能
- 只新增页面

输出：
完整代码 + 说明

---

## 3. 推荐 Prompt 模板

目标：
做什么？

上下文：
项目情况

功能：
具体需求

限制：
能改什么？不能改什么？

输出：
要什么结果？

---

# 🤖 二、Agent（代理）

## 1. 什么是 Agent

Agent = 会“执行任务”的 AI

---

## 2. 和普通 AI 区别

普通AI：回答问题  
Agent：执行任务  

---

## 3. Agent 能做什么

- 写代码
- 改代码
- 修Bug
- 读项目
- 做分析

---

## 4. 如何用 Agent（关键）

👉 先让 AI 思考，再执行

---

### 示例

第一步：

先不要写代码，分析如何新增评论功能

第二步：

按方案实现代码

---

# 🧠 三、Skill（技能）

## 1. 什么是 Skill

Skill = AI 的能力模块

---

## 2. 常见 Skill

### 页面生成

生成 React 页面

---

### Debug

修复报错

---

### 代码解释

讲解代码逻辑

---

### 后端开发

写 API / Flask

---

## 3. 推荐 Skill

1. 页面生成 Skill  
2. Debug Skill  
3. 代码解释 Skill  
4. API Skill  

---

# 🔗 四、三者关系

你 → Prompt → Agent → Skill → 结果

---

# 🎯 五、完整案例

## Prompt

新增评论功能

## Agent

分析 → 修改代码 → 完成任务

## Skill

- 写代码
- 数据库
- API

---

# 🚀 六、大创建议

- 写好 Prompt
- 用 Agent 执行
- 用 Skill 提升质量

---

# 🔥 最后总结

Prompt 决定上限  
Agent 决定执行  
Skill 决定能力
